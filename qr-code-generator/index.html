Description

Online tool to generate QR Code image. It supports:

  • Text
    Enter any text and the corresponding QR code will be generated instantly
  • URL
    Enter a valid URL starting with the http or https protocol and the corresponding QR code will be generated instantly
  • Email
    Enter a recipient's email address along with the subject and body of the email and the corresponding QR code will be generated instantly
  • WiFi
    Select the Authentication Type from the list, such as WEP, WPA, WPA2 or nopass, and then enter the SSID and passphrase if any and the corresponding QR code will be generated instantly

You can download the QR code image bu clicking the download button that will appear once you hover the QR code image
